# Changelog

All notable changes to Atletas World will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-09

### Added
- Initial release of Atletas World platform
- User authentication system with email verification
- Athlete profile management with photo uploads
- Sport categories and disciplines
- Achievement and badge system
- Event management functionality
- Django admin interface customization
- GitHub Pages demo site
- Responsive Bootstrap 5 UI

### Technical
- Django 5.0 backend
- SQLite database (development)
- REST API foundation
- Static file management with WhiteNoise
