Payments & Stripe

Live ✓

How clients pay

Clients go to My Portal → Packages and click Purchase. A payment modal opens with Apple Pay, Google Pay, or card. Once paid, the package activates automatically.

Issuing refunds

Owner Portal → Payments → find the payment → click Refund. Full or partial refunds are supported.

Viewing revenue

Owner Portal → Finances shows monthly revenue, YTD totals, pending payments, and rental income.

Stripe keys: Stored in /var/www/atletasworld/.env on the server. Never share the secret key (rk_live_...) over email or text.
📧

Email Notifications

Setup required

Sending a notification blast

Owner Portal → Notify → choose a recipient group → write subject + message → Send.

  • All Clients — everyone with an APC account
  • Unregistered Contacts — past event attendees who haven't signed up yet
  • Contacts by Source — e.g. only APC Summer 2025 attendees
  • Individual Select — hand-pick specific people

Enabling real email delivery

Email currently prints to the server terminal only. To enable delivery, add these lines to /var/www/atletasworld/.env and restart:

EMAIL_BACKEND=django.core.mail.backends.smtp.EmailBackend
EMAIL_HOST=smtp.gmail.com
EMAIL_PORT=587
EMAIL_USE_TLS=True
EMAIL_HOST_USER=info@atletasperformancecenter.com
EMAIL_HOST_PASSWORD=xxxx xxxx xxxx xxxx   ← Google App Password
PRODUCTION_EMAIL_ENABLED=True

Generate an App Password at myaccount.google.com/security → App Passwords (requires 2-Step Verification to be ON).

Clients & Contacts

Registered Clients

  • → Clients — view all accounts, approve/reject coach & renter access
  • → Players — all registered athletes
  • → Waivers — annual liability waiver status per client
Coaches and Facility Renters require manual approval before they can access their features. Check Clients for pending approvals.

Imported Contacts

Past event attendees (395 parents, 534 players) imported from program registration forms.

  • → Contacts — browse, search, edit, filter by source
  • → Automatically linked to a client account when they register with the same email
  • → Use Notify → Unregistered Contacts to invite them to create an account
📦

Packages & Credits

Managing packages

  • → Packages — create, edit, archive, duplicate
  • → Set sessions included, validity weeks, price
  • → Mark as Special Event Package for time-limited programs (camps, clinics)
  • → APC Select Membership type grants auto-credits + discounts on booking

APC Select Credits

  • → Credits — view and grant $40/month credits
  • → When a client purchases an APC Select package, 6 × $40 credits are auto-created
  • → Pick-up sessions: auto-discounted to $5 for Select members
  • → Camps & Clinics: 10% off automatically applied at booking
📅

Sessions & Schedule

Session Types (Catalog)

  • → Sessions — define what types of sessions are offered
  • → Set name, format, duration, price, drop-in price, max participants
  • → Toggle Show in Events / Show in Programs for homepage display
  • → Archive to hide without deleting; Restore to bring back

Coach Schedule

  • → Coaches → Schedule button — view and manage a coach's availability
  • → Each slot is a ScheduleBlock — date, time, session type, max participants
  • → Clients book directly from these blocks in their portal

Bookings

  • → Bookings — full list of all bookings, filterable by status and date
  • → Statuses: Pending → Confirmed → Completed / No Show / Cancelled
  • → Clients can cancel up to 24 hours before; after that contact the owner
🏟️

Field Rentals

Managing rental services

  • → Services — define rentable spaces (Full Field, Partial Field, Room, Gym)
  • → Set price per slot and whether approval is required

Approving rental requests

  • → Rentals — see pending requests (badge on nav shows count)
  • → Approve or Reject each request — client is notified automatically
  • → Create rental slots in advance to make them available for booking
🏋️

Coaches

  • → Coaches — add, edit, deactivate coaches; view session counts per coach
  • → Each coach has their own portal at /coach-portal/ to manage their schedule, players, and assessments
  • → Schedule button — view a coach's full calendar and add/edit their availability blocks
  • → Coach accounts are created here (not via self-signup). Set username + temporary password, then share credentials with the coach
📋

Waivers

  • → Waivers — see which clients have signed the annual liability waiver and which haven't
  • → Clients are required to sign before they can book any session
  • → Waivers reset on January 1 each year — clients sign again in the new calendar year
  • → Full audit log: name, typed signature, IP address, date/time for each signature
  • → Coaches and staff are exempt from the waiver requirement
🏆

Teams

  • → Teams — view all teams, their coaches, and player rosters
  • → Teams are created by Team Coach clients from their portal
  • → Team Coach accounts need approval — check Clients for pending requests
  • → Team Training packages unlock the team booking flow for team coaches

Deploying updates

Push code to main branch on GitHub → GitHub Actions runs tests → automatically deploys to the server. No manual steps needed for code changes.
